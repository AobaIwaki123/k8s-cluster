🚀 発展

以下のリポジトリとProxmoxを組み合わせることで、VMの作成・削除、構成の自動化が可能になり、自宅に簡易的なクラウド基盤を構築できます。

  • Terraform for Proxmox - インフラのコード化
  • Ansible - 構成管理の自動化
